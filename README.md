# PulseX

PulseX is a **social and marketplace mobile application** built with **React Native & Expo**, powered by **Firebase**.  
It allows users to connect, chat, share posts, buy/sell items, watch live streams, and enjoy entertainment content.

## Features

- **Authentication:** Email/password login & register (Firebase Auth)
- **Home Feed:** Display posts and activity
- **Forum:** Public & private forums for sharing
- **Marketplace:** Users can advertise, buy, and trade items
- **Chat:** Direct messaging and friend requests
- **Live:** Placeholder for live streaming
- **Entertainment:** Placeholder for videos and reels
- **Profile:** User profiles, followers, and settings
- **Navigation:** Bottom tab navigation for easy access to all screens
